# 🌿 API Ateliê Verdanza

API REST para o e-commerce de plantas Ateliê Verdanza.

**Stack:** Node.js • Express • JWT • bcrypt • Zod • Helmet

---

## 🚀 Como rodar

```bash
# Instalar dependências
npm install

# Iniciar servidor (produção)
npm start

# Iniciar com nodemon (desenvolvimento)
npm run dev

# Servidor roda em:
http://localhost:3001
